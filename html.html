CSS

Now it is time to explore CSS! This groovy language will add some style baby!
CSS stands for cascading style sheets.

This text is a dark pink color.

That text was dark pink! But how?
By using CSS and assigning the tag <pink> to that color text,
anything with that tag has the variables assigned to it in the css script!

Here is the code that did it:

pink {
 color: #E75480
}

I will break this down. By assigning the <pink> tag to have a color of #E75480, aka pink,
any text with this flag will now have that property of pink.
