How to Run LargeBass:

Windows:

To run LargeBass on Windows 11/10 64 bit os, follow the steps below.


Firstly, make sure you have installed LargeBass. Unzip the folder, and transfer it to where you want to store it. I would recommend storing it in Documents.

Look up "Thony", go to their webpage, install it, and set it up.

Start up Thonny. In Thonny, open the .py file in the "LargeBass" file. Press the run button.

Expand Thonny's shell, and use it to interact with LargeBass.
